（1）MyEclipse中添加JDK与绑定Tomcat遇到的问题及解决方法

myeclipse2017JDK的添加和Tomcat的绑定

1、在MyEclipse里绑定JDK1.8步骤：

（1）点击Windows → preference→JAVA→Installed JREs，点击add ，选择JDK的安装路径，然后勾选上就行了。

（2）设置JDK版本
点击Windows → preference → Java →Compiler
然后选择相应的版本就行。

2、添加Tomcat
（1）点击Windows → preference →server—>
Runtime environment—>add—>选择Tomcat的安装路径。找到server的窗口，然后右键，new，server，添加Tomcat的版本
