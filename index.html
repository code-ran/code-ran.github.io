（4）windows 10必装的五款神器

​Windows10软件推荐

@author:code-ran

（1）火绒安全软件 🤞

官网地址
介绍：
这是一款Windows下的安全杀毒软件，但是它的功能却不仅限于杀毒和保护系统安全。最让用 户称赞的地方是它的：弹窗拦截的功能以及开机自启动的自定义管理。也许你的电脑开机很慢，而且总是有许多烦人的弹窗广告。那么火绒这款软件，一定是你不二选择。有了它，你将不用再忍受弹窗广告的痛苦。

（2）bandzip 👍

官网地址
介绍：
这是一款韩国发明的解压缩软件，解压和压缩的速度非常快。而且还是免费、清亮、无广告的，实在是很良心。

（3）potplayer视频播放器❤

官网地址
这款视频播放神器几乎支持所有格式的视频，而且还有超过4倍速的设置，以及它强大的硬件解码技术，一流的补帧技术，而且里面还有很多个性化的设置。并且免费、无广告。如果你遇到一个视频无法解码播放，那你可以用它试试，相信它绝对不会让你失望的。

（4）天若OCR😊

官网地址
你还在为提取图片中的文字而烦恼？你是否遇到过百度文库里的文字无法复制的情况。或者图片中的公式无法写出来的情况。所有的这些问题，天若OCR都能帮你轻松搞定，它能精确的识别图片、视频、PDF、中的文字进行提取，你可以轻松的复制。获取你想获取的任何界面的内容。而且它还支持公式识别，这一点做得真的很不错。

（5）bandicam 😁

官网地址
一款致力于高清录屏的软件，你可以设置导出的视频的清晰度。每次录屏最长免费十分钟，😂十分钟后自动结束录屏。若想无线录屏升级到VIP即可，个人觉得十分钟已经够用。
上面的所有软件我都会放在编程小牛的公众号里，回复Windows10 五款高效软件即可获取下载链接，大家也可以到上面给的官网链接去下载。

超级详细的JDK与Tomcat的安装及配置教程

1. JDK的安装与环境配置

首先我们需要去官网上下载JDK对应的版本，以我现在要下载的1.8为例。
（1）官网：JDK官网地址
具体操作如下




然后下载之后去下载路径下去安装即可，我下载好了放在了我的C盘的Program Files /java下
注意：然后安装也是安装在这个路径下。安装JDK的同时也会自动安装JRE。

（2）配置JDK的环境
步骤：右键我的电脑，属性，高级系统设置，环境变量，
然后在系统变量里新建变量名为：JAVA_HOME然后点击浏览目录，选择JDK安装的路径；

然后新建变量名：CLASSPATH ，变量值： .;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar

之后找到系统变量的path,然后双击，然后点击新建，添加：%JAVA_HOME%\bin并将其移到最顶端。然后逐一确定。

（3）检查我们是否配置成功
win+R,输入cmd
然后在输入java，他会弹出一串东西。

然后再输入：java -version
会有如下提示说明我们安装成功了。

2、Tomcat的安装与环境配置*

**我们去官网下载Tomcat，这里以我自己下载的Tomcat7为例。**
[Tomcat官网地址（点击此处即可访问）](http://tomcat.apache.org/)

来到官网后，点击官网的archives下载需要的版本；

选择我们要下载的Tomcat7；

选择我们要下载的tomcat7的版本；

选择bin文件夹；

选择对应的版本和需要的种类免解压版或者压缩版进行下载。

接下来我们需要配置环境变量
在系统变量里新建：CATALINA_HOME，然后点击浏览目录，选择我们解压安装的Tomcat路径
我的是在c盘的根目录下。然后点击确定。

然后添加%CATALINA_HOME%\bin 然后点击确定。

然后我们要去到Tomcat的安装路径下找到conf路径下的登录页面记事本。

打开之后将该处的utf-8该成GBk 然后保存即可。

然后最后我们要测试Tomcat是否配置成功；
win+R,cmd，然后输入：startup.bat

（1）MyEclipse中添加JDK与绑定Tomcat遇到的问题及解决方法

myeclipse2017JDK的添加和Tomcat的绑定

1、在MyEclipse里绑定JDK1.8步骤：

（1）点击Windows → preference→JAVA→Installed JREs，点击add ，选择JDK的安装路径，然后勾选上就行了。

（2）设置JDK版本
点击Windows → preference → Java →Compiler
然后选择相应的版本就行。

2、添加Tomcat
（1）点击Windows → preference →server—>
Runtime environment—>add—>选择Tomcat的安装路径。找到server的窗口，然后右键，new，server，添加Tomcat的版本
