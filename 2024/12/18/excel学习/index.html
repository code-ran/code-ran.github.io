学习版本

Office2024

Excel应用场景

  1. 记录整理
  2. 数据计算
  3. 数据分析
  4. 数据展现

Excel自定义功能区

在功能区进行右键，选择自定义功能区。

选择新建选项卡，添加自定义的组，将常用命令添加到自定义的组当中，就可以在功能区显示新增的自定义功能区内容。

Excel保存设置密码

工具 —-> 常规选项

(1)设置了打开权限密码，没有密码则无法打开。

(2)设置了修改权限密码，无法对源文件进行修改操作。

Excel版本概述

目前Excel有两个大的版本: Excel2003和Excel2007及以上的版本

格式 扩展名 特征 结构 单sheet最大数据量
Excel 97 - Excel 2003工作簿 xls 存储容量有限 二进制格式，核心是复合文档结构 行: 65535 列: 256
Excel 工作簿 xlsx 基于xml压缩、占有率小，效率高 xml类型结构 行: 1048576 列: 16384

Excel快速填充

在B1输入jack之后，使用快捷键ctrl+e或者在开始中选择填充-快速填充功能。

Excel快速分析

选中需要分析的表格区域，右下角会出现快速分析的图标，点击之后可进行操作。

格式化中的快速分析操作，可以在开始—->条件格式中找到。

Excel数据录入的正确方式

按行录入，tab跳到下一个单元格,enter跳到下一行。(减少鼠标的点击)

特殊内容录入

（1）日期、时间: excel中可以识别以’ - ‘或者’ / ‘分割的日期，以’ : ‘分割的时间

快速获取当前的日期快捷键: ctrl + ;

快速获取当前的时间快捷键: ctrl + shift + ;

快速获取当前的日期+时间,使用now函数单元格键入: **=now() **

回车即可

(2)分数

正常输入1/5会显示1月5日的日期。需要输入0然后空格再输入对应的分数

(3)过长数据

比如身份证一般是18位的，但是excel的数据精度只有15位，超出15位会以科学计数法的格式进行数据显示。

解决方案为:

（4）原位填充

ctrl+enter

多个Excel合并到一个Excel

多个Excel在同一个目录下，点击数据—->获取数据—->来自文件—->从文件夹

数据导入

如何数据自网站进行的导入，还可以选中单元格区域: 数据 —-> 全部刷新 —->连接属性对数据的实时刷新频率进行设置。

点击查询 —-> 编辑使用Power Query插件对实时导入的数据进行编辑，例如列名的编辑。

数据刷新后，列名就变成了自定义的列名

表格转换成超级表

ctrl + t快捷键可以将普通表格转换成超级表

表设计-转换为区域可以将超级表还原为之前的普通表格。

转换为超级表之后，可以使用power query进行编辑

单元格格式

单元格内容

在单元格内填充的内容，可以分为三种基本数据类型

1.文本

默认左对齐，文本不可以参与计算。

2.数值

默认右对齐，数值计算精度是15位。

3.逻辑值

默认居中对齐，True和False既可以用来判断，也可以参与计算。

单元格格式

快捷键: Ctrl + 1

函数

在单元格中输入= 然后输入对应的函数 —->tab，选择参与函数计算的单元格—->enter即可。

COUNT函数(只统计数值类型)

利用COUT函数可以计算单元格区域中数字字段的输入项个数。

![GIF 2024-12-21 14-17-33](excel学习/GIF 2024-12-21 14-17-33.gif)

COUNTA函数(统计所有类型的数据，不包含空值)

利用函数COUNTA可以计算单元格区域包含数据的单元格个数。

COUNTBLANK函数(统计所有类型的数据，包含空值)

利用函数COUNTBLANK可以计算单元格区域包含空格的单元格个数。

COUNTIF(range,criteria)

单条件计数,输入=COUNTIF函数，选择单元格区域，点击fx()，输入对应的条件即可。

COUNTIFS（）

多条件计数

SUM()

求和，使用sum函数或者使用**alt + =**进行快速求和

AVERAGE()

求平均数

Ctrl + E快速填充

Ctrl + D向下填充

将选定区域内最顶层单元格的内容和格式复制到下面的单元格中.

Ctrl + 1

快速调出设置单元格的对话框

Ctrl + B

加粗字体

Ctrl + I

设置斜体

Ctrl + U

设置下划线

Ctrl + H

替换

Ctrl + T

创建超级表

Ctrl + \

Alt + =
